# Flux Cuts

Flux Cuts landing page built with Next.js.

## Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Build

```bash
npm run build
```

## Structure

- `app/page.tsx` composes the landing page.
- `app/components/` contains the page sections.
- `app/content.ts` contains section content, links, media URLs, and the Flux Cuts logo URL.
- `app/globals.css` contains the responsive styling.
